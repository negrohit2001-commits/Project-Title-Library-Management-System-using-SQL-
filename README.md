# Project-Title-Library-Management-System-using-SQL-
Design and develop a Library Management System using SQL. The project should involve three tables: Books, Members, BorrowingRecords. The system will manage book inventories, member details, and borrowing transactions

This project demonstrates a complete SQL solution for a simple Library Management System, including database setup, data insertion, and advanced retrieval and reporting queries.
The primary goal is to manage books, members, and borrowing transactions, including identifying overdue books, tracking borrowing activity, and generating usage reports.
